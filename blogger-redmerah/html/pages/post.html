Perbedaan List, Tuple, dan Set pada Python

Duis Occaecat

Pada bahasa Python, terdapat tiga jenis struktur data yaitu List, Tuple, dan Set. Meskipun memiliki tujuan yang sama, yaitu menyimpan banyak data dalam satu variabel, ketiga jenis ini memiliki perbedaan dalam cara penggunaan dan fungsionalitasnya.

Sebelum membahas perbedaannya, mari kita jelaskan secara singkat apa itu List, Tuple, dan Set. Jika Anda ingin langsung melihat tabel perbedaannya, silakan klik di sini.

Penjelasan Singkat List, Tuple, dan Set

List, Tuple, dan Set merupakan jenis tipe data iterable dalam Python yang digunakan untuk menyimpan kumpulan data secara simultan. Masing-masing dapat menyimpan berbagai jenis data, termasuk nested array. Data-data tersebut dipisahkan oleh koma.

Terdapat perbedaan dalam cara penulisan ketiga jenis struktur data ini:

Veniam irure culpa cupidatat nulla id fugiat sint consectetur nulla ea pariatur exercitation.

Excepteur nulla tempor officia aute ad do incididunt ea et anim ad.

Sit Lorem duis reprehenderit sunt est.

Nulla magna exercitation aliqua nisi nulla laborum sunt irure aute cillum amet consequat.

  • List ditandai dengan tanda kurung siku [ ].
  • Tuple ditandai dengan tanda kurung biasa ( ).
  • Set ditandai dengan tanda kurung kurawal { }.

Berikut adalah contoh penulisan ketiga jenis struktur data tersebut:

# Contoh `List`
myList = [1, 'a', True, [3, 4]]

# Contoh `Tuple`
myTuple = (1, 'a', True, [3, 4])

# Contoh `Set`
mySet = {1, 'a', True}

Perbedaan pada Set

1. Urutan data dalam Set bersifat acak

Urutan data dalam Set tidaklah penting, yang penting adalah keberadaan datanya.

Contoh:

mySet = {'a', 'b', 'c'}
print(mySet)

Hasil eksekusi program di atas dapat menghasilkan output berikut:

> py main.py
{'a', 'b', 'c'}

> py main.py
{'b', 'a', 'c'}

> py main.py
{'c', 'b', 'a'}

Karena sifat acak ini, Set tidak dapat diakses menggunakan operator [].

Contoh:

myList = ['a', 'b', 'c']
data_pertama_list = myList[0]  # ok: urutan pertama adalah 'a'

mySet = {'a', 'b', 'c'}
data_pertama_set = mySet[0]  # error: urutan dalam `Set` bersifat acak

2. Set tidak menyimpan data yang sama

Set mirip dengan himpunan, sehingga jika terdapat data yang sama, hanya satu data tersebut yang akan disimpan.

Contoh:

mySet = {1, 2, 3, 1, 1, 1}
print(mySet)

# Output: {1, 2, 3}

3. Set hanya dapat menyimpan data yang dapat di-hash

Set hanya dapat menyimpan data yang dapat di-hash, seperti string, integer, dan sejenisnya. Data yang tidak dapat di-hash, seperti iterable (List/Tuple/Set), tidak dapat dimasukkan ke dalam Set.

Contoh:

mySet = {1, 2, 3}
# Ok: `1`, `2`, `3` dapat di-hash

mySet2 = {1, 2, [1, 2, 3]}
# Error: `[1, 2, 3]` tidak dapat di-hash

Untuk mengetahui apakah suatu data dapat di-hash atau tidak, Anda dapat menggunakan fungsi bawaan hash().

Contoh:

hash_value_int = hash(1)
# Ok: `1` dapat di-hash

hash_value_list = hash([1, 2, 3])
# Error: `[1, 2, 3]` tidak dapat di-hash

Perbedaan pada Tuple

1. Tuple bersifat tidak dapat diubah (immutable/read-only)

Tuple memiliki sifat tetap dan tidak dapat diubah atau dihapus. Jika suatu data terdapat dalam Tuple, maka data tersebut akan tetap ada dan tidak dapat diubah. Bahkan urutan data dalam Tuple tidak dapat diubah.

Contoh:

myTuple = (1, 2, 3)
print(myTuple[0])  # ok: membaca data tanpa mengubahnya

myTuple[0] = 3  # error: `Tuple` tidak dapat mengubah data di dalamnya

Perbedaan pada List

List memiliki fungsionalitas yang lebih luas dibandingkan Set dan Tuple. List dapat melakukan semua yang tidak dapat dilakukan oleh Set dan Tuple. Dengan demikian, List tidak memiliki perbedaan khusus seperti Set dan Tuple.

Kesimpulan

  • List adalah struktur data terurut yang dapat melakukan semua yang tidak dapat dilakukan oleh Set dan Tuple.

  • Tuple adalah struktur data terurut yang bersifat tetap (immutable/read-only), sehingga data dalam Tuple tidak dapat diubah.

  • Set adalah struktur data tidak terurut yang tidak peduli dengan adanya data yang sama. Data dalam Set haruslah data yang dapat di-hash.
